# 📱 blacklun's Portfolio
> **Secure, Local-first, and Data-Driven Mobile Solutions.**

안녕하세요, 사용자의 데이터를 안전하게 관리하고 시각화하는 앱 개발자 **blacklun**입니다.

## 🔒 Security Philosophy
1. **Zero Server Transmission**: 사용자의 모든 데이터는 기기 내부(Local)에만 머뭅니다.
2. **Robust Encryption**: SQLCipher를 활용하여 로컬 DB 파일 자체를 암호화합니다.
3. **Data Mobility**: 사용자가 자신의 데이터를 언제든 백업(CSV)하고 제어할 수 있는 권리를 보장합니다.
